Задание 6 

Ошибка заключалась в том что нельзя без ограничения использовать конструкцию new T() компилятор не будет знать 
какой тип будет передан. Решается проблема добовление ограничения при объявлении обобщенного класса

Вот пример исправленного кода:
public class Instantiator<T> where T : new()
{
    public T instance;
    
    public Instantiator()
    {
        instance = new T();
    }
}
